# Chat System Installation

Before getting started, insert the latest version of this product from our [Discord](/socials/discord) into your game.

>>> Insert your file
Insert chat-system.rbxm into Roblox Studio.
>>> Placement & Positioning
Place it inside workspace.
!!!info
Any misplaced models **will not** work, keeping them in their respective folders is required.
!!!
>>> Done
Head on over to the [configuration](/Chat/Config) to set its behavior!
>>>
